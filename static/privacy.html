BlitzScrim

Privacy

BlitzScrim analyzes World of Tanks Blitz replay files to generate session statistics.

Uploaded files

Replay files are uploaded to the server only for analysis. They are written to a dedicated temporary folder and deleted after processing completes or fails.

Stored data

The service does not intentionally store replay files, player lists, exports, or generated summaries after the request is finished.

Processing

Statistics are generated server-side with wotbreplay-inspector. The default summary is generated locally from the computed stats and is not sent to an external AI provider.

Limits

Uploads are limited by file count, total request size, temporary session count, and temporary storage lifetime to keep the public service stable. When the analyzer page is closed, the browser also notifies the server so stale temporary storage can be cleaned.
